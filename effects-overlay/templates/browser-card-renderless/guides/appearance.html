見た目を変える

多層カードの見た目はほぼ style.css で調整します。カード構造を変える場合だけ index.html も触ります。

最初に触る場所

カード幅

--card-width と .browser-card-list を見ます。

角丸

--card-radius が外側と内側の丸みに効きます。

余白

--card-gap、.card-content、.compact-card を調整します。

テーマ色

[data-kh-theme] ごとの --theme-a/b/c を変えます。

例: カードを細くする

.browser-card-list {
  width: min(var(--card-width, 440px), calc(100vw - 24px));
}

設定 UI から変えたい場合は manifest.json の cardWidth default も揃えます。

例: 左レールを消す

"toggleCssVars": {
  "showMetaRail": ["--rail-display", "flex", "none"]
}
.meta-rail {
  display: var(--rail-display, flex);
}

例: typing を速くする

.typing-char {
  animation: typing-rise 180ms ease forwards;
  animation-delay: min(calc(var(--char-index) * 14ms), 520ms);
}

注意点

  • カードの幅や padding を大きく変えたら、OBS の推奨幅も見直します。
  • 背景レイヤーは .card-backdrop と .card-shell の2層です。
  • .is-paid-pulse は JS の afterBindComment() で一時的に付きます。
