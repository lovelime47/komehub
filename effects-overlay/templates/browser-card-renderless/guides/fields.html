表示項目を増やす

表示項目は index.html の <template id="comment-template"> に追加します。

data-kh の考え方

data-kh="name" は model.name、data-kh="display.notice" は model.display.notice を読みます。

<span class="name" data-kh="name"></span>
<div class="notice" data-kh="display.notice" data-kh-empty="hide"></div>
<div class="comment-text" data-kh="display.html" data-kh-mode="html"></div>

data-kh-empty="hide" の意味

値が空のときに要素ごと非表示にします。role、notice、amount、avatar、badge、sticker のようにコメントごとに有無が変わる項目へ付けます。

<span class="role-pill" data-kh="display.role" data-kh-empty="hide"></span>

例: コメント時刻を追加する

HTML に表示場所を追加する

<div class="sub-line">
  <span class="theme-label" data-kh="display.themeLabel"></span>
  <span class="time" data-kh="display.timeText" data-kh-empty="hide"></span>
  <span class="amount" data-kh="amount" data-kh-empty="hide"></span>
</div>

JS で値を作る

display: {
  timeText: rawComment.timestamp || '',
  html: buildTypingHtml(rawComment, prevModel),
  role: roleLabel(rawComment),
  notice: noticeLabel(rawComment),
  themeKey: themeKey,
  themeLabel: THEME_LABELS[themeKey] || 'LIVE'
}

CSS を追加する

.time {
  color: color-mix(in srgb, var(--theme-a) 70%, #344);
  font-size: 0.72em;
}
