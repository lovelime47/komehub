設定 UI を増やす

設定は manifest.json に UI を追加し、index.html の JSON config か script.js callback へつなぎます。

3つのつなぎ方

CSS 変数

幅、文字サイズ、角丸、色など。cssVars に書きます。

toggle

表示/非表示や compact 切替。toggleCssVars または toggleClasses に書きます。

callback

テーマ固定のように JS の状態を変える設定。config.callbacks に書きます。

例: カード幅 (width) が CSS 変数へ流れる仕組み

表示位置 (左/右) と横幅 (width) は全テンプレ共通の設定として自動で付きます。このテンプレートは共通の width をカード幅として --card-width に流しています。

{
  "key": "width",
  "type": "slider",
  "label": "カード幅",
  "min": 340,
  "max": 760,
  "default": 520,
  "suffix": "px"
}
"cssVars": {
  "width": ["--card-width", "px"]
}

例: typing ON/OFF

"toggleClasses": {
  "typingEnabled": "typing-enabled"
}
#comments:not(.typing-enabled) .typing-char {
  opacity: 1;
  transform: none;
  animation: none;
}

例: テーマ固定

themeMode: function (value) {
  var mode = String(value || 'rotate');
  var nextTheme = 'theme-' + mode;
  fixedTheme = mode === 'rotate' || !THEME_LABELS[nextTheme] ? '' : nextTheme;
}
