表示項目を増やす

コメント1件の中に何を表示するかは index.html の <template id="comment-template"> で決めます。 値の差し込みには data-kh を使います。

data-kh の考え方

runtime はコメント1件ごとに「表示用のデータのかたまり」を作ります。 このページではそれを model と呼びます。model は raw comment の値と、 beforeCommitComment() が追加した値を合わせたものです。 

{
  "name": "fuwari-voice",
  "comment": "👏👏👏👏👏",
  "amount": "",
  "display": {
    "prefix": "",
    "html": "<span class=\"typing-char\">👏</span>..."
  }
}

data-kh は、この model のどの場所を HTML に入れるかを指定する目印です。 data-kh="name" なら model の一番上にある name を読み、 data-kh="display.html" なら display の中の html を読みます。 

<span class="name" data-kh="name"></span>


<div class="comment-text" data-kh="display.html" data-kh-mode="html"></div>

点で区切った path は、入れ子の object をたどるという意味です。 display.html は model.display.html、 display.prefix は model.display.prefix です。

data-kh-empty="hide" の意味

data-kh-empty="hide" は、data-kh で読んだ値が空のときに その要素ごと隠す指定です。コメントによって有ったり無かったりする項目に付けます。 

<span class="amount" data-kh="amount" data-kh-empty="hide"></span>

この例では、スパチャ金額があるコメントなら amount を表示し、 金額がない普通のコメントなら .amount 要素ごと非表示にします。 

model.amount = "¥1,000"  -> 表示する
model.amount = ""        -> 要素ごと隠す
model.amount = null      -> 要素ごと隠す
model.amount が無い      -> 要素ごと隠す

これを付けない場合、値が空でも要素は残ります。 その結果、空の余白、空のラベル枠、壊れた画像アイコンのような見た目になることがあります。

例: コメント時刻を追加する

HTML に表示場所を追加する

<div class="comment-meta">
  <img class="badge" data-kh="badge" data-kh-empty="hide" alt="">
  <span class="name" data-kh="name"></span>
  <span class="time" data-kh="display.timeText" data-kh-empty="hide"></span>
  <span class="amount" data-kh="amount" data-kh-empty="hide"></span>
</div>

JS で値を作る

return {
  display: {
    timeText: new Date(rawComment.timestamp || Date.now()).toLocaleTimeString('ja-JP', {
      hour: '2-digit',
      minute: '2-digit'
    }),
    prefix: prefix,
    html: buildTypingHtml(rawComment, prevModel)
  }
};

CSS で見た目を整える

.time {
  flex: 0 0 auto;
  color: var(--muted-color);
  font-size: 0.66em;
}

例: アバターを完全に外す

設定で非表示にするだけなら showAvatar が使えます。 テンプレートとして完全にアバターを持たない設計にするなら HTML と CSS の両方を変えます。

HTML

<img class="avatar" ...> を削除します。

CSS

.comment {
  grid-template-columns: 6px minmax(0, 1fr);
}

注意点

  • HTML だけ追加しても、model に値がなければ空になります。
  • 空のときに非表示にしたい要素には data-kh-empty="hide" を付けます。
  • HTML として差し込みたい場合だけ data-kh-mode="html" を使います。
