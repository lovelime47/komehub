structure / usage

配信者向け設定

このテンプレートを OBS に載せるときに最初に触る設定です。 CSS や JS を読まなくても、配信画面に合わせた見た目へ調整できる範囲をまとめています。

見た目 設定 UI 表示項目 元データ加工 一時効果

最初のおすすめ

OBS サイズ

幅は 460px 前後、高さは 650px 前後から始めます。コメント欄を大きく見せたい場合は、まず フレーム高さ を上げます。

表示位置

表示位置 で左下、中央下、右下を選びます。ゲーム画面や字幕とぶつからない位置を選びます。

文字サイズ

1080p なら 19px 前後が基準です。720p やスマホ視聴を重視する場合は 21px 以上にします。

背景に合わせる

暗い配信画面

標準 または クール を選び、背景濃度は 0.75 から 0.9 の範囲で調整します。

明るい配信画面

モノクロ または 暖色 を選び、背景濃度を 0.9 以上にします。本文の色は白系のままが読みやすいです。

雑談の雰囲気を強く出す

ヘッダーの ヘッダー上の小ラベル と ヘッダータイトル を配信名に合わせます。

コメント量に合わせる

状況触る設定目安
コメントが少ない最大表示数5 から 8
コメントが多い最大表示数10 から 15
詰まって見えるコメント間隔12px から 18px
情報量を増やしたいコンパクト表示ON

タイピング演出

タイピング演出 はコメント本文を 1 文字ずつ出す設定です。 雑談では ON のままが自然ですが、コメント量が多い配信では OFF にすると読みやすくなります。

速度の考え方

タイピング速度 は次の文字を出し始めるまでの間隔です。数字を小さくすると速く、数字を大きくするとゆっくりになります。
