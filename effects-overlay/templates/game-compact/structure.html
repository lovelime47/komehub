game-compact 構造解説

game-compact はゲーム配信のデフォルトテンプレートであり、 Render 実装の最小サンプルです。複雑な独自処理を持たず、 runtime.starters.list() に描画先、コメント行テンプレート、設定の接続だけを渡します。

ファイルの役割

ファイル役割
index.html#comments だけを置く最小の入口です。
script.jsコメント行の <template> を作り、runtime に登録します。
style.cssゲーム画面を隠しにくい、背景なしの軽い見た目を定義します。
manifest.json設定 UI に出す項目とデフォルト値を定義します。

最小構造

<div id="comments"></div>
<script src="script.js"></script>

HTML にコメント行を直接書かないのがポイントです。 1 コメント分の DOM は script.js の <template> で作ります。

data-kh の意味

属性入る内容
data-kh="avatar"投稿者アイコン画像。
data-kh="badge"メンバーバッジなど。
data-kh="name"表示名。
data-kh="amount"スパチャ金額。
data-kh="membershipHeader"メンバー加入などの見出し。
data-kh="text"コメント本文。YouTube カスタム絵文字もここに入ります。
data-kh="sticker"ステッカー画像。

設定の流れ

manifest.json の uiSchema にある key を、 script.js の config.cssVars / toggleCssVars / toggleClasses へ接続します。 CSS はその変数や class を読むだけにしておくと、Render のサンプルとして理解しやすくなります。

新しい最小テンプレートを作る場合は、このテンプレートを複製し、 manifest.json の ID / 表示名と、style.css の見た目から変更してください。
