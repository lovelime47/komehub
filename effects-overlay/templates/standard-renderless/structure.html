standard-renderless 構造解説

最小の renderless 縦リストを理解するための短い地図です。 詳しい教材は framed-list-renderless/、共通仕様は本体のテンプレート開発ガイドを参照してください。

ファイルの役割

index.html

#comments と <template> を持ち、data-kh で model の値を表示します。

script.js

htmlFirst.start() を呼び、beforeCommitComment() で display.html を作ります。

style.css

コメントカード、名前、本文、入退場状態の最小スタイルを定義します。

manifest.json

表示件数、文字サイズ、色など、アプリの設定画面に出す項目を定義します。

コメント表示の流れ

rawComment.comment
  -> script.js の beforeCommitComment()
  -> display.html
  -> index.html の data-kh="display.html"
  -> style.css の .comment-text

最初に触る場所

見た目だけ変える

style.css の .comment、.name、.comment-text を触ります。

表示項目を増やす

index.html の <template> に data-kh 要素を追加します。

表示用の値を作る

script.js の beforeCommitComment() に display.* を追加します。
