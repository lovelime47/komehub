structure / appearance

見た目を変える

横流れの見た目は style.css の #comments、#track、.comment、glow 関連 class を中心に変えます。

最初に触る場所

縦位置

#comments bottom は positionY callback で上書きされます。

横間隔

#track gap を変えるとコメント同士の距離が変わります。

光彩

--stroke-width、--glow-core、--blur-main を applyGlow() が更新します。

名前位置

.name-bottom class で名前行を本文の上/下へ切り替えます。

例: コメント間隔を狭くする

#track {
  gap: 24px;
}

例: スパチャをさらに大きくする

.comment.is-superchat {
  font-size: 1.25em;
}

.is-superchat は runtime が amount > 0 のコメントに自動で付けます。

例: 光を弱くする

.text, .amount, .sticker, .membership-badge {
  filter:
    drop-shadow(0 0 0.05em var(--glow-core))
    drop-shadow(0 0 0.25em var(--glow-main));
}
