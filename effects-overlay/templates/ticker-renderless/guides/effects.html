横流れの動きを変える

ticker の動きは、CSS の #track transition と runtime の track 計算が組み合わさって決まります。

このサンプルで既にやっていること

travelSeconds: ANIM_SPEEDS.normal,
transitionTiming: 'cubic-bezier(0.12, 0, 0, 1)'
#track {
  transition: transform var(--anim-duration, 0.8s) cubic-bezier(0.12, 0, 0, 1);
}

速度を変える

function setTravelSeconds(helpers, seconds) {
  var ticker = helpers.ticker || helpers;
  if (ticker && ticker.setTravelSeconds) ticker.setTravelSeconds(seconds);
}

直接 CSS だけを変えるのではなく、runtime の setTravelSeconds() を通すと内部状態と揃います。

再計算が必要な変更

  • 文字サイズを変える
  • 名前を上/下へ切り替える
  • 表示項目を増やして横幅が変わる
  • gap や padding を大きく変える

これらは recalcTicker(helpers) を呼ぶ候補です。

overflow trim の注意

コメントが増えすぎたとき、見えている左端のコメントをすぐ消すと「動き始めに消えた」ように見えます。runtime は画面外に出たコメントから整理するため、テンプレート側で無理に DOM を削除しないでください。
