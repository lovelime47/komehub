表示項目を増やす

横流れでも表示項目の追加方法は list と同じです。<template id="ticker-item-template"> に data-kh を置きます。

data-kh の考え方

<span class="name" data-kh="name"></span>
<span class="amount" data-kh="amount" data-kh-empty="hide"></span>
<span class="text" data-kh="text"></span>

membershipHeader や amount は runtime の標準 model にあるため、beforeCommitComment() なしで直接読めます。

data-kh-empty="hide" の意味

値が空のときに要素ごと隠します。横流れでは空要素が残ると横幅計算にも影響するため、任意項目には付けます。

<img class="badge" data-kh="badge" data-kh-empty="hide" alt="">

例: 時刻を追加する

HTML に追加する

<span class="time" data-kh="timestamp" data-kh-empty="hide"></span>

CSS を追加する

.time {
  margin-right: 0.35em;
  font-size: 0.72em;
  opacity: 0.75;
}

必要なら再計算する

表示幅が変わる設定を追加した場合は、callback 内で recalcTicker(helpers) を呼びます。
