structure / settings

設定 UI を増やす

横流れでは、設定変更がコメント幅や track 移動距離に影響するかを必ず見ます。

3つのつなぎ方

CSS 変数

文字色、文字サイズ、フォントなど。cssVars でつなぎます。

class

名前表示、バッジ表示、メンバー色分けなど。toggleClasses でつなぎます。

callback

縦位置、速度、光彩、再計算が必要な設定。script.js に処理を書きます。

例: 名前位置を増やす

{
  "key": "namePosition",
  "type": "buttons",
  "options": [
    { "value": "top", "label": "上" },
    { "value": "bottom", "label": "下" }
  ]
}
namePosition: function (value, _config, helpers) {
  helpers.container.classList.toggle('name-bottom', value === 'bottom');
  recalcTicker(helpers);
}

例: 速度 preset を増やす

var ANIM_SPEEDS = { slow: 1.2, normal: 0.8, fast: 0.5, instant: 0.25 };
{ "value": "instant", "label": "超速" }

確認ポイント

  • 横幅や高さが変わる設定では recalcTicker(helpers) を呼ぶ。
  • 速度変更は setTravelSeconds() を使う。
  • JSON config 内にはコメントを書かない。
