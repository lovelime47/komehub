グロー色を増やす

このテンプレートの色替えは theme asset ではなく、光彩用の RGB と CSS 変数で行います。

色のつながり

manifest glowColor / glowCustomColor
  -> script.js hexToRgb()
  -> applyGlow()
  -> --glow-core / --glow-main
  -> style.css の text stroke / drop-shadow

例: purple を追加する

JS に追加する

var GLOW_COLORS = {
  pink: [255, 100, 180],
  cyan: [0, 220, 255],
  purple: [170, 120, 255]
};

manifest に追加する

{ "value": "purple", "label": "パープル", "hex": "#aa78ff" }

初期値を変える場合

"glowCustomColor": "#aa78ff",
"glowColor": "purple"

光の強さとの関係

glowIntensity は色ではなく、縁取り太さ、透明度、ぼかし量を変えます。色を変えたいときは glowCustomColor、強さを変えたいときは glowIntensity です。
